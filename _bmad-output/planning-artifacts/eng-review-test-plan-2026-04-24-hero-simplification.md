## Key Interactions to Verify

- **Hero CTAs** : "Choisir mon appartement" scroll vers `#logements`, "Voir les photos" scroll vers `#photos` (comportement inchangé)
- **Cartes Airbnb** : bouton "Réserver sur Airbnb" sur chaque carte redirige vers le bon URL (1ch / 2ch / 3ch). Vérifier qu'aucun bouton n'a été cassé par le rename des `name` fields
- **Navigation** : les liens de nav inchangés, "Maison Ayaba" apparaît toujours dans le logo / footer (branding préservé)
- **Sticky mobile CTA** : vérifier texte cohérent (à inspecter, non-auditée dans ce review)

## Edge Cases

- **Viewport 375px** (iPhone SE) : le hero doit tenir dans un seul écran (H1 + subtitle + 2 CTAs, pas de scroll nécessaire pour voir les CTAs)
- **Viewport 414px** (iPhone Pro Max) : idem
- **Viewport 768px** (iPad portrait) : pas de régression desktop
- **Langue EN** : tout le texte traduit, pas de clé i18n brute affichée (ex: "neighborhood.offer1ch")
- **JS désactivé** : le hero reste lisible (contenu statique Astro, devrait être OK)
- **Lecteur d'écran** : `alt` des images cohérent avec les nouveaux noms ("Salon de l'appartement 1 chambre")

## Critical Paths

1. **Test des 10 secondes (assignment du design doc)** : 3 visiteurs mobile différents, chacun 10 secondes sur la homepage. Objectif : 2/3 minimum expliquent correctement "c'est un appartement à Cotonou, on peut prendre 1, 2 ou 3 chambres selon le besoin."
2. **Parcours de réservation complet** : visiteur arrive sur homepage → clique sur carte 2 chambres → atterrit sur l'annonce Airbnb `wabi-sabi-cotonou-2ch`. Aucune rupture.
3. **Cohérence cross-pages** : visiteur qui navigue homepage → /fr/appartement → /fr/fidjrosse-cotonou ne doit jamais voir "Duo/Famille/Tribu" nulle part.

## Regression Guard

- **SEO** : comparer Google Search Console 14 jours après deploy. Keywords "appartement Cotonou Fidjrossè" doivent rester top 3.
- **Vercel Analytics** : taux de clic sortant vers airbnb.fr doit être stable ou en hausse 7 jours après deploy.
- **Build** : `npm run build` doit passer sans warning. `grep -r "offerDuo\|offerFamille\|offerTribu\|Logement Duo\|Logement Famille\|Logement Tribu" dist/` doit retourner 0 résultat.
